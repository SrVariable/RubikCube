# RubikCube

## Description

Simple project to play with a [Rubik's Cube](https://en.wikipedia.org/wiki/Rubik%27s_Cube).

## Requirements

- git
- gcc
- make

## Usage

Clone the repository

```Shell
git clone https://github.com/SrVariable/RubikCube.git
```

Go to the cloned repository
```Shell
cd RubikCube
```

Execute make

```Shell
make
```

Run the program

```Shell
./rubik
```
